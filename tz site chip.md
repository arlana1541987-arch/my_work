# Техническое задание для Cursor AI

  


## Задача

  


Разработать современный SEO-оптимизированный сайт компании по чип-тюнингу автомобилей. В качестве ориентира использовать структуру сайта [https://chip-tyuning-spb-78.ru](https://chip-tyuning-spb-78.ru), но полностью переработать дизайн, код и интерфейс. Не копировать HTML, CSS, изображения и тексты.

  


---

Использовать:

  


* Next.js 15 (App Router)

* TypeScript

* TailwindCSS

* Shadcn UI

* Framer Motion

* React Hook Form

* Zod

* Lucide Icons
